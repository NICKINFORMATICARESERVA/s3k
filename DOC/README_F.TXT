
        *** Sonic and Knuckles Collection ***

-----------------------------------------------------------------------------

Configuration minimum de système
    *Windows 95
    *Pentium 75MHz ou CPU supérieure
    *16 Megabytes de mémoire libre
    *Minimum de 20 Megabytes de disque dur de libre

Conifiguration de sytème recommandé
    *Pentium 133MHz ou CPU supérieure

Veuillez insérer le CD Sonic and Knuckles Collection dans le lecteur de CD 
pour jouer.

-------------------------------------------------------------------------------

Notes:

1. Il vous faut DirectX pour jouer à Sonic and Knuckles Collection. DirectX 
est installé automatiquement quand le program setup est exécuté.Si
pour une raison ou autre, DirextX n'est pas installé, veuillez l'installer en
utilisant dxsetup.exe qui se trouve dans le directoire DirectX du CD.

2. Il est possible que vous aurez des difficultés en jouant en mode true-colour 
en fenêtre. Si ceci est le cas, jouez au jeu en mode plein écran (F4).

3. Si vous débranchez le jostick ou changez de joystick pendant la partie, 
quittez le jeu et reconfigurez les propriétés du jostick.

4. Le mode plein écran créera peut-être des problèmes graphiques sur certaines 
machines. Si ceci est le cas, vérifiez l'écran "Check this box if you experience
graphical problems" dans les options de l'écran de choix de jeu.
